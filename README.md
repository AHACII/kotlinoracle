# SmartHomeAR 🏠📱

**SmartHomeAR** is an innovative Android application that combines Augmented Reality (AR) with Home Design. It allows users to visualize furniture and decor in their own space before making a purchase.

## 🚀 Features

### 👤 User Side
- **Interactive Home:** Explore recommended products and categories (Salon, Chambre, Bureau, etc.).
- **AR Catalogue:** Browse a wide range of furniture with advanced filtering.
- **AR Visualization:** Place 3D models of furniture in your real environment using the camera.
- **Project Management:** Save your favorite designs and room layouts.
- **Smart Scanner:** Scan your room to prepare for AR placement.

### 🛠 Admin Panel
- **Product Management:** Add new products with names, prices, images, and categories.
- **Inventory Control:** Delete and manage existing furniture in the catalogue.

## 🛠 Tech Stack
- **Kotlin** & **Jetpack Compose** (Modern UI)
- **SceneView / ARCore** (Augmented Reality)
- **Retrofit & GSON** (API Communication)
- **Navigation Compose** (Screen Routing)
- **Coroutines** (Asynchronous tasks)
